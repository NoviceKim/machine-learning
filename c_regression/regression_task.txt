범주형 데이터 쓸 거면 이진(종류 2가지)인 것으로 연습
OLS, VIF는 실제 데이터 예측 평가와 최대한 맞게

저번처럼 Cycle 돌려보기
분포 조정은 맨 마지막 수단으로


- 사용 데이터 세트 -
1. 전복 특징(성별, 길이, 부위 별 무게 등)에 따른 나이 예측 - 8th Cycle 진행 중
2. 각 합금 별 피로지점 예측 - 수치형 컬럼 27개
3. 콘크리트 강도 예측 - 수치형 컬럼 9개


- 과정 -
선형 모델 먼저해보고 만족할만한 결과 나왔을 때 다음 거 진행
